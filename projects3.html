Context

For a class project we were assigned with the task of creating an interactive interface that filters, sorts and aggregates favorites from a dataset. My chosen interface was a Dating Profile Manager that allows users to go through matches and select favourites. The interface allows users to go through their matches and add favourites to the match-list.

Problem

How can we create an interface that filters, sorts and aggregates data interactively?

Dating Development

Dating Development is a simplified dating match sorter and aggregator

  • The users may filter through age, location or interests of the person
  • facilitates easy navigation
  • high color contrast and readability throughout their design
  • eye-catching visuals and unique branding

Competitive Analysis

I conducted a competitive analysis of popular variations of selection interfaces

Hulu allows users to navigate items to watch and add to ‘My Stuff’ for easy access or ‘liking’ it to see similar content. The aggregator does not organise the data through a specific filter.

Amazon allows users to navigate through the shopping portal and add to cart or add to a list for later and allows for duplication in the cart

Instagram save allows for content to be saved in custom categories or the general collection to be viewed later. This is sorted chronologically.
